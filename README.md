# Student Management Database - Task 2

This project extends the Student Management Database by adding Courses and Enrollments tables. It uses SQLite to manage student enrollments, course grades, and performance analysis.

Features:
- Create Courses and Enrollments tables
- Store student grades for each course
- List students enrolled in courses
- Calculate average grade per course
- Find top 3 performing students
- Count students who failed (grade < 40)

Technologies Used:
- SQLite
- SQL
